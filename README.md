# isa-ios-objc

通用分层,易测试,灵活iOS架构

## 优点?
1.高内聚，低耦合  
2.MVVM 表现形式 (ObjC ReactiveCocoa Swfit版本 RxSwift，Android版本 RxJava,RxKotlin)  
3.多环境任意切换  
4.无API开发(可本地模拟API)  
5.单元测试(已有部分)
